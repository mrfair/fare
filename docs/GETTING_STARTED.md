> *fare: route-first vanilla SPA toolkit for jQuery/vanilla devs — no VDOM, no rerender headaches.*

---
# Getting Started / เริ่มต้นใช้งาน

## Install
```bash
npm i
```

## Dev
```bash
npm run dev
```

## Build & Preview
```bash
npm run build
npm run preview
```

## Project structure
- `src/app/` core runtime (router, dom wrapper, lifecycle)
- `src/routes/` routes (folder-based)
- `src/components/` optional reusable components

> **Tip:** migrate ทีละหน้า (route) ก่อน แล้วค่อยแยก component ทีหลัง
